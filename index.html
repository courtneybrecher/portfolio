Battleship Game – Functional Programming

Collaborated on a functional reimagining of the classic Battleship game using OCaml, emphasizing data abstraction and immutability. Themed around a mermaid lagoon, the game includes both two-player and single-player modes with difficulty-scaled AI, point tracking, and terminal-based interactions.

  • Built gameplay logic for firing, hit detection, sunk treasures, and win conditions
  • Developed comprehensive test suites and input error handling
  • Implemented point system and user commands like view and quit
  • Created a clean TUI with flushed screen turns and visual hit/miss feedback
  • Extended game with AI ship-placement for easy, medium, and hard difficulty

Interactive Image Selection

This project focused on implementing and optimizing the Intelligent Scissors algorithm for interactive image segmentation, allowing users to select and trace paths that follow image edges for precise selections. I implemented Dijkstra’s algorithm to find the shortest paths in a weighted graph, adapting it for incremental updates to visualize the selection process dynamically. This approach enabled real-time feedback, allowing users to adjust paths and control points interactively. The final tool seamlessly combines edge-following and straight-line path options, offering flexibility and high accuracy in selections. With background processing and real-time updates, the tool maintained a smooth, responsive user experience. The project highlighted advanced algorithm design, data structure optimization, and real-time system performance.

Cloudvirga Dashboard Design

One of my key projects involved collaborating with the design team to redesign Cloudvirga’s platform, aligning it with the company’s updated branding, values, and mission. A primary goal of the redesign was to encourage users to complete their home loan applications. I played an integral role in every stage of the design process, from participating in regular design reviews to revising Figma files to improve the user interface. I also ensured the site met ADA compliance standards, creating a more inclusive experience for all users.
